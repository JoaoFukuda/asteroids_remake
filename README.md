# Asteroids Remake
Jogo baseado em Asteroids, mas com algumas modificações como:
* Janelas dinâmicas
* Suporte para multiplayer (no futuro)
* Modos co-op e versus
* Várias opções e modificações diferentes para você escolher
* Muito, (não) muito mais...
